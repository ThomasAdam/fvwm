DestroyMenuStyle

   --------------------------------------------------------------------------

   DestroyMenuStyle menustyle

   Deletes the menu style named menustyle and changes all menus using this
   style to the default style, you cannot destroy the default menu style.

 DestroyMenuStyle pixmap1
