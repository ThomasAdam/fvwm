:doctype: manpage


NAME
----

fvwmutils - the Fvwm utils meta-man page.


SYNOPSIS
--------

fvwm-[utility] [options]


DESCRIPTION
-----------

Fvwm utils are shell or perl scripts to make Fvwm life easier. See the *OVERVIEW* 
section for more information.


OVERVIEW
--------

In this meta-man page all utilities shipped with Fvwm are described short in 
alphabetical order and followed afterwards in detail.

* *fvwm-bug* - report a bug in fvwm
* *fvwm-config* - query information of an existing fvwm installation
* *fvwm-convert-2.4* - convert fvwm 2.2.x configuration file to fvwm 2.4 syntax
* *fvwm-convert-2.6* - convert fvwm 2.4.x configuration file to fvwm 2.6 style
* *fvwm-menu-directory* - builds a directory browsing menu for fvwm
* *fvwm-root* - Sets the root window of the current X display to image
