GnomeShowDesks

   --------------------------------------------------------------------------

   GnomeShowDesks number

   This command tells GNOME compliant applications the number of desktops
   that are used by fvwm.

   --------------------------------------------------------------------------

   fvwm 2.6.3 (from cvs)
