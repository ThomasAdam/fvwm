NAME
----

fvwmmodulecommands - Commands affecting modules.


SYNOPSIS
--------

DESCRIPTION
-----------

fvwmmodulecommands - Details the commands used to manipulate how to start
external Fvwm modules, how it affects their behaviour, etc.
