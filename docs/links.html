The Official FVWM Homepage - Links

Here's a few other sites (besides the official one) that have FVWM info:
  • Scott Boyd's Configuration Samples page
  • Pager Balloon patch
  • Bob Woodside's Miscellaneous patches
  • Barry Warsaw's Python/FVWM pages
And here's a few others that have info on programs that are designed to work with FVWM in some manner:
  • The Dotfile Generator Home Page - A program that helps you generate .emacs, .cshrc, .fvwmrc, etc.
  • John Lapeyre's FvwmConf
  • The TkGoodStuff Home Page - A Tcl/Tk Program that performs like GoodStuff/FvwmButtons.
And here's some window managers that were originally created from fvwm source, but PLEASE DON'T ASK QUESTIONS ABOUT THEM ON THE FVWM LISTS! Rather, contact the parties responsible for them directly:
  • SCWM - The Scheme Configurable Window Manager
  • Bowman - A NeXT Lookalike created from 1.24r (link no longer exists?)
  • AfterStep - A continuation of Bowman
  • Fvwm95 - A hack of an fvwm2 beta that looks more like (shudder) Win95
  • Fvwm decorations - A collection of fvwm screenshots and config files
  • Fvwm decorations - Even more fvwm screenshots and config files
And here's an excellent site with all sorts of window manager information and comparisons (new location):
  • Window Managers for X
